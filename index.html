O que são sistemas distribuídos?

Sistemas distribuídos na teoria

Antes de entendermos o que são sistemas distribuídos é necessário entender anteriormente dois conceitos essenciais para a plena compreensão, esses são rede de computadores e protocolos. As redes de computadores podem ser definidas como infra-estrutura servindo um conjunto de computadores interconectados através de links de comunicação de topologias e mídias diversas. Protocolos podem ser definidos como um conjunto de normas que permitem que duas ou mais máquinas conectadas à internet se comuniquem entre si. Funciona como uma linguagem universal, que pode ser interpretada por computadores de qualquer fabricante, por meio de qualquer sistema operacional. Dessa forma, um sistema distribuído é composto de vários computadores se comunicando através de uma rede de computadores, onde computadores abrigam conjuntos de processos que se comunicam através de um conjunto de protocolos distribuídos para assistir a execução coerente das atividades distribuídas.
